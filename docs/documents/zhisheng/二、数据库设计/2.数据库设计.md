## 1）功能模块分类表

表名：func_type

| 字段名 | 字段类型     | 说明         |
| ------ | ------------ | ------------ |
| id     | int          | 主键、自增   |

表名：func_module

| 字段名 | 字段类型     | 说明                                  |
| ------ | ------------ | ------------------------------------- |
| id     | int          | 主键、自增                            |
| name   | varchar(255) | 功能名，unique                        |

| type   | int          | 外键：功能模块分类（func_type - id）  |
