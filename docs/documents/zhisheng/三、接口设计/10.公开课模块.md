## 查询公开课分类

> 请求地址：/course/getCourseType
>
> 请求方式：POST

**响应** 

1. 成功

   ```json
   {
       "status": 200,
       "data": [
           {
               "id": 1,
               "name": "计算机",
               "isdel": 0
           },
       ]
   }
   ```

2. 失败

   ```json
   {
       "status": 0,
       "message": "还没有分类，请联系管理员添加"
   }
   ```
   



## 查询分类下的课程

> 请求地址：/course/getCourse
>
> 请求方式：POST

| 请求参数 | 类型   | 说明         |
| -------- | ------ | ------------ |
| tid      | Number | 公开课分类ID |

**响应**
