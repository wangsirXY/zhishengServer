## 获取社团活动信息

> 请求地址：/club/activity
>
> 请求方式：POST

| 请求参数 | 是否必填 | 类型 | 说明   | 备注                             |
| -------- | -------- | ---- | ------ | -------------------------------- |
| cid      | False    | int  | 社团ID | 默认值：0 代表获取所有的活动信息 |

                       "present": 1,

## 新建社团活动

> 请求地址：/club/addActivity
>
> 请求方式：POST

| 请求参数 | 是否必填 | 类型   | 说明         | 备注                                             |
| -------- | -------- | ------ | ------------ | ------------------------------------------------ |
| name     | True     | String | 社团活动名称 |                                                  |
| message  | True     | String | 社团活动介绍 |                                                  |
| time     | True     | String | 活动开始时间 | 时间格式尽量为 2022-1-21<br />或者 new Date()    |
| cid      | True     | Number | 社团ID       |                                                  |
| state    | False    | Number | 活动状态     | 0：未开始 / 1：进行中 / 2：已结束<br />默认值：0 |

**响应** 

1. 成功

   ```json
   {
       "status": 200,
       "message": '创建社团成功！'
   }
   ```

2. 失败

   ```json
   {
       "status": 0,
       "message": '创建社团活动失败，检查必填字段是否完整！'
   }
   ```
